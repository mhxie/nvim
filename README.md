## Personal Neovim Configuration

My personal neovim configuration for daily coding.

Supported Language: C/C++, Rust, Python3

## Prerequisites

### Basic

* neovim 0.5+
* vim-plug

### Language Servers

* rust-analyzer

## Installation

    cd ~/.config
    git clone https://github.com/mhxie/nvim.git
    nvim
    :PlugInstall
